## Selected Projects

### Hedera-py-lite  
Lightweight Python SDK for interacting with Hedera services without heavy dependencies.  
Repository: https://github.com/De-real-iManuel/hedera-py-lite

### ZK-PIN  
Zero-knowledge PIN verification system proving a 4-digit secret without revealing it.  
Built with Circom, SnarkJS, and Poseidon hashing.  
Repository: https://github.com/De-real-iManuel/ZK-PIN

Decentralized “Bouncer API” using jury consensus to validate localized human data and reduce AI hallucinations.  
Repository: https://github.com/De-real-iManuel/rem-protocol

Architecture for transparent electricity billing using AI + blockchain infrastructure.
